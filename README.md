# Tip Calculator

A simple **Android Tip Calculator App** built using **Kotlin** and **Jetpack Compose**.  
This project demonstrates handling user input, state management, and modern Android UI development with Compose. It helps users calculate tip amounts based on a bill and tip percentage.

## About

The Tip Calculator app is an Android application built as a learning project to practice Kotlin and Jetpack Compose. Users can input a bill amount, select a tip percentage, and view the calculated tip. It highlights core Android development skills and Compose UI patterns.

## Features

- Enter the **bill amount**
- Choose or input a **tip percentage**
- Display **calculated tip value**
- Optional **rounding** for tip
- Simple and responsive UI using Jetpack Compose
